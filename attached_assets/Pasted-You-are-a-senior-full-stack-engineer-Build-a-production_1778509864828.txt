You are a senior full-stack engineer.

Build a production-ready Progressive Web App (PWA) called "Kopi Tiang Alam".

# PROJECT OVERVIEW

Kopi Tiang Alam is a modern coffee shop ordering system where customers can order food and drinks without registration or login.

Customers can:

* Scan QR code on table for dine-in
* Input their name for takeaway orders
* Browse menu
* Add products to cart
* Checkout orders
* Track order status in realtime

Admin can:

* Login securely
* Manage products
* Manage categories
* Manage tables
* Generate QR codes
* Manage customer orders
* Update order statuses

The system must be mobile-first, responsive, modern, and optimized for coffee shop usage.

# CUSTOMER FLOW

## DINE IN FLOW

1. Customer scans QR code on table
2. System automatically detects table number
3. Customer sees menu
4. Customer adds items to cart
5. Customer checkout order
6. Order sent to admin dashboard
7. Customer tracks order status

Example URL:
https://kopitiangalam.com/menu?table=07

---

## TAKEAWAY FLOW

1. Customer opens website
2. Customer inputs name
3. Customer browses menu
4. Customer checkout order
5. System generates queue number
6. Customer tracks order status
